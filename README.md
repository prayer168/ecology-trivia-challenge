# 環境知識競賽大挑戰

這是一個單頁互動答題網頁，包含 3 種難度、共 60 題環境知識競賽與自然生態、環保知識選擇題。

## 使用方式

開啟 `index.html`，或在此資料夾啟動靜態伺服器後瀏覽：

```powershell
python -m http.server 8765
```

接著前往 `http://127.0.0.1:8765/`。

## 教學建議

- 可作為自然課的暖身、複習或分組搶答活動。
- 每題作答後會顯示簡短解析，適合引導學生討論「生物之間的關係」。
- 題目主題包含森林、昆蟲、海洋、土壤、濕地、微生物、食物網、城市生態、空氣品質、水污染、資源回收、淨零與環境政策。
- 難度分成「簡易」、「中度難度」、「超難度」，每組 20 題。
- 每題作答後會有鼓勵文字與短音效提示。

## 功能

- 難度切換：簡易、中度難度、超難度。
- 隨機練習：可隨機出題並打亂選項順序。
- 計時模式：可切換 10 分鐘倒數挑戰。
- 音效開關：可依教室情境開啟或關閉提示音。
- 答題報告：完成後顯示分數、百分比與各主題表現。
- 錯題複習：完成後列出錯題解析，並可只重練錯題。
