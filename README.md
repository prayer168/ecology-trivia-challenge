# 環境知識競賽大挑戰

互動式環境、生態與環保知識競賽網頁。題庫分成 5 個等級，共 100 題：

- 初階：20 題
- 入門：20 題
- 進階：20 題
- 魔王等級：20 題
- 出神入化：20 題

## 使用方式

在專案資料夾啟動本機伺服器：

```powershell
python -m http.server 8765
```

開啟：

```text
http://127.0.0.1:8765/
```

## 功能

- 五種等級切換，每個等級 20 題。
- 題目順序每次自動隨機。
- 選項順序每次自動隨機。
- 每輪 20 題的正確答案位置平均分散，第 1 到第 4 個選項各 5 題。
- 答題後顯示答對或答錯回饋、正確答案與解析。
- 支援計時模式、音效、主題表現報告與錯題複習。

## 檔案

- `index.html`：頁面結構與難度按鈕。
- `styles.css`：版面、按鈕與 RWD 樣式。
- `script.js`：題庫、出題、隨機、計分、回饋與錯題複習邏輯。
- `assets/ecology-hero.png`：首頁視覺圖。
